# Simple Shortcut

A tiny, fast text expander for Windows. Type a short abbreviation or press a key combination — Simple Shortcut instantly replaces it with any text you want: greetings, e-mail templates, addresses, code snippets, anything.

<img width="800" height="450" alt="preview" src="https://github.com/user-attachments/assets/43e98f73-b3bc-44ed-9890-eab140f9bd25" />

## Features

- **Text shortcuts** — type shortcuts like `hello` anywhere and it turns into your full text the moment you finish typing it
- **Key binds** — assign a hotkey like `CTRL+ALT+B` to paste a text instantly
- **Lightweight** — one small exe in the system tray, no background services, no dependencies
- **Autostart** — optional start with Windows, toggled from the tray menu

## How it looks

### Tray menu

Simple Shortcut lives in your system tray. Right-click the icon to open the settings or toggle autostart.

### Creating a shortcut

Give your shortcut a text trigger (like `hello`), a key bind (press the keys you want — it's recorded automatically), or both. Then enter the text it should expand to.

## Installation

**Installer:** download `SimpleShortcutSetup.exe` from the [Releases](../../releases) page and run it. It installs to `Program Files\Simple Shortcut` and lets you choose Start Menu and desktop icons.
